表單處理

表單依照傳送方法屬性(method="get"或method="post")分為GET表單處理與POST表單處理，GET表單將表單資料附加於網址列方式進行傳送，傳送大小最多約2K Bytes，POST表單將表單資料置於HTPP內容中傳送，接下來將針對GET表單與POST表單進行說明。

GET表單處理-$_GET

GET表單中表單屬性method必須設為get，表單欄位必須設定 name 屬性，後端PHP程式可藉由 表單欄位 name 屬性值來取得該表單欄位值，使用者點選送出GET表單後，表單資料會利用網址列的方式(xxx.php?欄位name值1=值1&欄位name值2=值2，傳送資料會以陣列方式儲存於 $_GET 的系統變數，使用者可使用 $_GET["表單欄位name值"] 來取得使用者傳送欄位值，請注意$_GET為系統變數，一定要大寫。以php-1.html呼叫php-1.php為例說明傳送使用者帳號與密碼

程式說明:前端表單欄位id屬性用於前端控制，以帳號為例，label標籤中for="f1"設定，該標籤將用於標籤屬性id="f1"的欄位，如果前端表單需要使用JavaScript，請記得設定存取欄位的id，送出按鈕value屬性用於設定按鈕顯示值，樣式類別class則套用Bootstrap。

php-1.html原始碼：


  <form method="get" action="php-1.php">
    <label for="f1">帳號</label>
    <input type="text" id="f1" name="account"><p></p>
    <label for="f2">密碼</label>
    <input type="text" id="f2" name="password"><p></p>
    <input type="submit" class="btn btn-primary" name="btn" value="送出">
  </form>

程式說明:後端程式php-1.php顯示$_GET陣列中account與password值。

php-1.php原始碼：


  <?php
    echo "帳號:".$_GET["account"]."<br/>";
    echo "密碼:".$_GET["password"]."<br/>";
  ?>

程式說明:前端表單欄位id屬性用於前端控制，以帳號為例，label標籤中for="f1"設定，該標籤將用於標籤屬性id="f1"的欄位，如果前端表單需要使用JavaScript，請記得設定存取欄位的id，送出按鈕value屬性用於設定按鈕顯示值，樣式類別class則套用Bootstrap。

表單欄位檢查函數-isset、empty

後端PHP程式接收表單資料後，在取得表單欄位值之前，必須要判斷該欄位是否存在或該欄位是否有值，請注意：如傳送未輸入文字欄位則該欄位沒有值("")，而核取方塊如為勾選，則該欄位不存在。因此後端PHP利用函數 empty 檢查沒有值欄位，利用 isset 檢查不存在的欄位。兩個函數傳回值均為布林值true或false

練習：請在前一練習表單中加入興趣核取方塊，並在後端php-1.php程式中檢查帳號、密碼或興趣欄位、是否有輸入或勾選
