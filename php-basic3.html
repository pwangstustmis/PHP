  • GET/POST表單處理-$_GET, $_POST

兩個網頁檔表單處理是指由一個表單檔案呼叫另一個表單處理PHP檔案，以下列程式為例，表單檔(p1.html或p1.php)呼叫處理檔案(p2.php)，請注意表單檔案可以是HTML或PHP檔，取決於表單檔案是否包含PHP動作，但是表單處理程式需為PHP檔。表單檔method可為get或post，用於設定資料傳送方式，而action屬性用於設定表單處理程式，下列程式設定傳送方法為get，並指定p2.php為表單處理程式，表單使用Bootstrap，請下載Bootstrap、解壓縮、並將解壓縮檔中bootstrap.min.css複製至網站根目錄中css檔案夾。請自行建立網站根目錄css檔案夾。

p1.html或p2.php原始碼


<html>
  <head>
    <meta charset="utf-8">
    <link rel="stylesheet" href="css/bootstrap.min.css"></link>
  </head>
  <body>
    <form method="get" action="p2.php">
      <div class="form-group">
        <label for="f1">帳號</label>
        <input type="text" class="form-control" id="f1" name="account">
      </div>
      <div class="form-group">
        <label for="f2">密碼</label>
        <input type="text" class="form-control" id="f2" name="password">
      </div>
      <button type="submit" class="btn btn-primary" name="btn">登入</button>
    </form>
  </body>
</html>

p2.php原始碼


<html>
  <head>
    <meta charset="utf-8">
    <link rel="stylesheet" href="css/bootstrap.min.css"></link>
  </head>
  <body>
  <?php
    echo "傳送帳號:".$_GET["account"]."<p>";
    echo "傳送密碼:".$_GET["password"]."<p>";
  ?>
  </body>
</html>

GET/POST表單處理-$_GET, $_POST

GET/POST表單處理取決於表單屬性method設定，如果method設為 get，則為GET表單，如果method設為 post，則為POST表單。不管是GET或POST表單，表單檔案中欄位必須設定 name 屬性，因為後端PHP程式可藉由表單欄位 name 屬性值來取得該表單欄位值。
使用者送出GET表單後，表單資料會利用網址列的方式(xxx.php?欄位name值1=值1&欄位name值2=值2 傳送，而傳送資料會以陣列方式儲存於PHP $_GET 的系統變數，使用者可以在PHP處理程式使用 $_GET["表單欄位name值"] 來取得使用者傳送欄位值，請注意$_GET為系統變數，一定要大寫。
如果使用者送出POST表單後，表單資料會經由HTTP request本體傳送至處理程式，而傳送資料會以陣列方式儲存於PHP $_POST 的系統變數，使用者可以在PHP處理程式使用 $_POST["表單欄位name值"] 來取得使用者傳送欄位值，請注意$_POST為系統變數，一定要大寫。先前為GET表單範例，下列程式表單檔(p1.htm)以POST方式呼叫處理檔案(p2.php)

p1.html或p2.php原始碼


<html>
  <head>
    <meta charset="utf-8">
    <link rel="stylesheet" href="css/bootstrap.min.css"></link>
  </head>
  <body>
    <form method="post" action="p2.php">
      <div class="form-group">
        <label for="f1">帳號</label>
        <input type="text" class="form-control" id="f1" name="account">
      </div>
      <div class="form-group">
        <label for="f2">密碼</label>
        <input type="text" class="form-control" id="f2" name="password">
      </div>
      <button type="submit" class="btn btn-primary" name="btn">登入</button>
    </form>
  </body>
</html>

p2.php原始碼


<html>
  <head>
    <meta charset="utf-8">
    <link rel="stylesheet" href="css/bootstrap.min.css"></link>
  </head>
  <body>
  <?php
    echo "傳送帳號:".$_POST["account"]."<p>";
    echo "傳送密碼:".$_POST["password"]."<p>";
  ?>
  </body>
</html>

表單陣列設定與處理

如果希望將表單欄位群組成一串傳送至後端PHP，則表單所有群組欄位 name 屬性必須設定為 name="欄位變數[]"，name="欄位變數[]" 設定會將這些群組欄位以陣列方式傳給PHP。下列範例讓使用者勾選興趣，興趣可以是多選，因此透過陣列方式來傳遞興趣欄位，請注意：PHP中 count($陣列變數) 會傳會陣列大小，陣列起始位置為 0，取得陣列方式為 $陣列變數[位置值])。

p1.html或p2.php原始碼


<html>
  <head>
    <meta charset="utf-8">
    <link rel="stylesheet" href="css/bootstrap.min.css"></link>
  </head>
  <body>
    <form method="post" action="p2.php">
      <div class="form-check form-check-inline">
        <input class="form-check-input" type="checkbox" id="chk1" value="逛街" name="hobbies[]">
        <label class="form-check-label" for="chk1">逛街</label>
      </div>
      <div class="form-check form-check-inline">
        <input class="form-check-input" type="checkbox" id="chk2" value="看電影" name="hobbies[]">
        <label class="form-check-label" for="chk2">看電影</label>
      </div>
      <div class="form-check form-check-inline">
        <input class="form-check-input" type="checkbox" id="chk3" value="美食" name="hobbies[]">
        <label class="form-check-label" for="chk3">美食</label>
      </div>
      <button type="submit" class="btn btn-primary" name="btn">登入</button>
    </form>
  </body>
</html>

p2.php原始碼


<html>
  <head>
    <meta charset="utf-8">
    <link rel="stylesheet" href="css/bootstrap.min.css"></link>
  </head>
  <body>
    <?php
    $hobbies=$_POST["hobbies"];
    echo "興趣個數:".count($hobbies)."<p>";
    foreach ($hobbies as $h)
    echo "興趣:".$h."<p>";
    ?>
  </body>
</html>

JavaScript非同步傳輸-fetch

當使用者執行 GET/POST 表單時，網頁控制會從現在表單檔跳至PHP檔，可是有時候 我們想要取得後端資源，可是希望網頁仍維持在現有的網頁，這個時候我們就需要非同步傳輸-fetch，非同步傳輸-fetch可將資料上載至後端PHP程式，或者呼叫後端PHP程式下載資料，針對後端資料下載，課程僅說明text與JSON資料，其他資料格式請自行搜尋。(請注意:fetch會傳回一個response的promise物件)。

p1.html原始碼-取得p2.php得傳回 Hello! 您好! 文字


<html>
  <head>
    <meta charset="utf-8">
    <link rel="stylesheet" href="css/bootstrap.min.css"></link>
    <script>
      function getText() {
        fetch('p2.php')
        .then(function(response) {
          return response.text();
        })
        .then(function(data) {
          console.log(data);
        });
      }
    </script>
  </head>
  <body>
  <button class="btn btn-primary" onclick="getText()">取得資料</button>
  </body>
</html>

p2.php原始碼


<?php
  echo "Hello! 您好!";
?>

p1.html原始碼-取得p2.php得傳回JSON


<html>
  <head>
    <meta charset="utf-8">
    <link rel="stylesheet" href="css/bootstrap.min.css"></link>
    <script>
      function getJson() {
        fetch('p2.php')
        .then(function(response) {
          return response.json();
        })
        .then(function(data) {
          console.log(data.school);
        });
      }
    </script>
  </head>
  <body>
  <button class="btn btn-primary" onclick="getJson()">取得資料</button>
  </body>
</html>

p2.php原始碼


<?php
  echo json_encode(array("school"=>"南臺科技大學"));
?>

使用非同步傳輸-fetch上載資料，課程使用JavaScript FormData物件來進行資料上載，新增FormData物件後，我們利用 document.getElementById("gender").value 取得下拉式方塊值，並在表單資料物件中新增 gender 欄位，之後透 post 方法並藉由 body 屬性將表單資料物件傳送至 p2.php。請注意：JavaScript利用id屬性取得控制項。後端p2.php會根據傳送值回傳答案。

p1.html原始碼-取得p2.php得傳回文字


<html>
  <head>
    <meta charset="utf-8">
    <link rel="stylesheet" href="css/bootstrap.min.css"></link>
    <script>
      function getText() {
        var fdata=new FormData();
        fdata.append('gender',document.getElementById("gender").value);
        fetch('p2.php',
        {
          method:'post',
          body:fdata
        }).then(function(response) {
          return response.text();
        })
        .then(function(data) {
          console.log(data);
        });
      }
    </script>
  </head>
  <body>
    <select id="gender">
      <option value="1">女</option>
      <option value="2">男</option>
    </select>
    <button class="btn btn-primary" onclick="getText()">取得資料</button>
  </body>
</html>

p2.php原始碼


<?php
  if ($_POST["gender"]==1)
    echo "Hello! 女!";
  else
  echo "Hello! 男!";
?>

表單欄位檢查函數-isset、empty

後端PHP程式接收表單資料後，必須要判斷該欄位是否存在或該欄位是否有值，請注意：如傳送未輸入文字欄位則該欄位沒有值("")，而核取方塊如過沒有勾選，則該欄位不存在。後端PHP使用函數 empty 檢查沒有值的欄位，使用 isset 檢查不存在的欄位。兩個函數傳回值均為布林值true或false

練習：請在前一練習表單中加入興趣核取方塊，並在後端p2.php程式中檢查帳號、密碼或興趣欄位、是否有輸入或勾選
