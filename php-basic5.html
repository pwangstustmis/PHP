檢查函數-empty、isset

isset函數於檢查變數是否存在，empty函數用於檢查變數是否為空值。


<?php
//變數$abs沒有宣告，因此輸出 變數不存在
if (isset($abc))
  echo "變數存在";
else
  echo "變數不存在";
$abc="";
//變數$abs為空值
if (empty($abc))
  echo "變數空值";
else
  echo "變數不是空值";
?>

字串函數-explode、htmlspecialchars、nl2br、str_repeat、strlen、substr

函數用於字串處理包含：

  • explode(分隔字元,字串)：將字串一分隔字元進行分割並傳回字串陣列。
  • htmlspecialchars(字串)：將字串中預設特殊符號如 <、>、"、'、& 轉為HTML編碼。
  • nl2br(字串)：將字串中跳行控制符號\n用HTML跳行標籤<br/>取代。
  • str_repeat(字串,重複次數)：重複字串。
  • strlen(字串)：傳回字串長度。
  • substr(字串,起始值,傳回長度)：傳回子字串，請注意：字串的第一個字起始值為0。

日期函數-date、time、date_create、date_diff

函數用於字串處理包含：

  • date("日期格式")：傳回設定格式日期字串，格式：d代表兩位數字日、D代表英文星期、m代表兩位數字月、M代表英文字月、Y代表4位數字年、y代表2位數字年。
  • time("時間格式")：傳回設定格式時間字串，格式：a代表小寫am/pm、A代表大寫AM/PM、h代表兩位數子12小時制、H代表兩位數子24小時制、i代表兩位數分、s代表兩位數秒。
  • date_create("日期時間字串")：傳回日期時間字串日期物件，如果沒有代入值，則傳回現在日期時間物件。
  • date_diff(日期物件1,日期物件2)：傳回兩個日期物件日期間距。

數學函數-abs、ceil、floor、max、min、pow、round

函數用於字串處理包含：

  • abs(數字)：傳回數字絕對值。
  • ceil(數字)：傳回小數無條件進位值。
  • floor(數字)：傳回小數無條件捨去值。
  • max(數字陣列)：傳回數字陣列最大值。
  • min(數字陣列)：傳回數字陣列最小值。
  • pow(底數,冪次方)：傳回底數的幾次方值。
  • round(數字,小數位)：傳回到小數位的四捨五入值。

轉址函數-header

轉址函數header用於進行轉址的動作，轉址命令後需搭配exit()或die()函數。

  • header("Location:檔案位址")：將執行頁面轉至對應檔案，請注意：轉址後需搭配exit()或die()函數。 


<?php
header("Location:myfile.php");
exit();
?>
