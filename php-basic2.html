表單標籤-form

表單最常為商業網頁使用，表單內容標籤需包含於form標籤內，form標籤常用屬性包含action、method與enctype，action屬性用於指定表單資料處理檔案，如未指定action則表單將傳送給自己，method屬性包含get與post值，當method屬性為get時，表單資料會透過網址列變數名稱與值的方式進行傳送，傳送長度約3000字大小，當method屬性為post時，表單資料會儲存於HTTP request內容中進行傳送，沒有長度限制。enctype屬性限用於method屬性值為post，用於設定表單資料編碼方式，一般為 application/x-www-form-urlencoded 或 multipart/form-data，前者為預設編碼方式，而後者用於上載檔案。

HTML原始碼：


<form method="get" action="負責處理表單php檔">

</form>
<form method="post" action="負責處理表單php檔">

</form>
表單輸入標籤-input

表單輸入標籤input是表單中使用最為頻繁的標籤，用於資料輸入，藉由type屬性設定text(文字)、button(按鈕)、color(顏色選取)、date(日期選取)、datetime-local(日期及時間選擇)、email(電子郵件輸入)、file(檔案選擇)、hidden(隱藏表單欄位)、image(圖片按鈕)、month(月份選擇)、number(數字選擇器)、password(密碼輸入)、range(範圍選擇器)、submit(送出按鈕)、tel(電話號碼)、time(時間選擇器)、week(月份選擇器)。屬性依據不同類型而有所不同，請參考MDN Wen DOcs Input element，如果input標籤一定要輸入則含屬性required ，要傳送給後端程式表單欄位請設定name屬性用於後端程式對應個別欄位，input標籤一般會搭配label標籤用於設定欄位標籤文字。

除上述類型之外，輸入標籤input表單還有包含checkbox與radio類型，checkbox設定核取方塊，用於多選，因此每個checkbox使用不同name屬性值，radio設定選項，用於單選，，因此每個radio使用相同name屬性值。

興趣 登山 購物

性別 女 男

HTML原始碼：


  <form>
    興趣 
    <input type="checkbox" value="1" name="h1">登山 
    <input type="checkbox" value="2" name="h2">購物<p></p>
    性別 
    <input type="radio" value="f" name="gender">女 
    <input type="radio" value="m" name="gender">男
  </form>
文字區域-textarea

textarea標籤用於設定多行文字輸入，屬性包含 rows(指定列數)與cols(指定每列字數)。

HTML原始碼：


  <form>
    <textarea cols="40" rows="5"></textarea>
  </form>
下拉式方塊-select

select標籤用於設定下拉式方塊選項，每個選項使用option標籤。

教育程度

HTML原始碼：


  <form>
    <label for="e1">教育程度</label>
    <select id="e1">
      <option value="1">高中(含)以下</option>
      <option value="2">大專</option>
      <option value="3">研究所以上</option>
    </select>
  </form>
