  1. PHP表單處理-一個網頁檔

內容涵蓋

  • PHP取得表單傳送方式- $_SERVER["REQUEST_METHOD"]

一個網頁檔表單處理是指 表單與表單處理檔案為同一個檔案 ，因此表單檔案為PHP。由於表單檔案與表單處理檔案為同一個檔案，因此表單 action 設定值為自己，請注意PHP有一系統變數 $_SERVER['PHP_SELF'] 會傳回顯在執行檔案的名稱，表單 action 請部要直接設 $_SERVER['PHP_SELF']，因為網頁 XSS 會攻擊這樣設定，使用者可直接設定檔名或使用 action="<?php =htmlentities($_SERVER['PHP_SELF']); ?>"。

p1.php


<html>
  <head>
    <meta charset="utf-8">
    <link rel="stylesheet" href="css/bootstrap.min.css"></link>
  </head>
  <body>
  <?php
    //使用者送出表單時執行
    if ($_SERVER["REQUEST_METHOD"]=="POST") {
        echo "使用者傳送帳號:"._POST["account"].", "."使用者傳送密碼:"._POST["password"];
    }
  ?>
    <form method="post" action="p1.php">
      <div class="form-group">
        <label for="f1">帳號</label>
        <input type="text" class="form-control" id="f1" name="account">
      </div>
      <div class="form-group">
        <label for="f2">密碼</label>
        <input type="text" class="form-control" id="f2" name="password">
      </div>
      <button type="submit" class="btn btn-primary" name="btn">登入</button>
    </form>
  </body>
</html>

PHP取得表單傳送方式- $_SERVER["REQUEST_METHOD"]

使用者點選表單傳送後，表單傳送方法 get或post 會設定於PHP系統變數 $_SERVER["REQUEST_METHOD"] 中，請注意：系統變數都必須是大寫，系統變數中值也是大寫，如果表單傳送方法為 get，系統變數 $_SERVER["REQUEST_METHOD"] 值為 GET，如果表單傳送方法為 post，系統變數 $_SERVER["REQUEST_METHOD"] 值為 POST，如果使用者沒有傳送表單，則$_SERVER["REQUEST_METHOD"] 預設值為GET。

練習：請用一個表單及POST方法傳送帳號密碼，預設帳號 abcde，密碼 12345，如果帳號或密碼錯誤，請在按鈕下方顯示錯誤訊息「帳號或密碼錯誤」。

練習：請用一個表單及GET方法傳送帳號密碼，預設帳號 abcde，密碼 12345，如果帳號或密碼錯誤，請在按鈕下方顯示錯誤訊息「帳號或密碼錯誤」。(請注意：檔案預設為GET傳輸模式，如何判斷使用者是否傳送表單？提示：可以使用 isset 函數)
