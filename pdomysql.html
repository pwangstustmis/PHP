使用資料庫之前，請記得啟動XAMPP控制台中MySQL選項。點選Admin使用PHPMyAdmin管理MySQL資料庫。

PDO 資料庫連接

PDO提供單一介面讓PHP程式來存取不同類型資料庫。使用資料庫之前，必須先建立資料庫連接，XAMPP MySQL預設帳號為root，沒有密碼。PDO("mysql:host=資料庫伺服器位置;dbname=資料庫名稱","使用者帳號","使用者密碼")會傳回對應資料庫的連接物件，使用者可使用下列程式建立資料庫連接物件： 


try {
    $pdo=new PDO("mysql:host=localhost;dbname=資料庫名稱;","root","");
} catch (PDOException $err) {
    die("資料庫無法連接");
}

練習：請使用PHPMyAdmin在MySQL建立iChef資料庫，並建立iusers資料表包含id、account、password、nickname、email、image等欄位，並手動輸入一筆資料。

練習：請使用PDO連接iChef資料庫。

PDO MySQL- 資料CRUD(Create、Read、Update、Delete)

PDO藉由prepare、execute命令來執行SQL語法。如果執行命令有代入參數，使用者可以在prepare命令中使用?代表執行時的代入參數，並在execute命令將參數以陣列方式代入。PDO prepare會傳回命令物件，該命令物件將用於excute(執行)SQL命令，並藉由物件fetch()、fetchAll()傳回單筆或所有資料，執行fetch()、fetchAll()時，可代入PDO::FETCH_NUM或PDO::FETCH_ASSOC，分別代表傳回答案陣列以數字或欄位名稱來對應陣列結果。

下列兩個select命令分別執行明細與群組，第一個命令包含下列關鍵字：

  • distinct：用於設定選取唯一值。
  • as：設定新欄位名稱。
  • inner join：設定關聯，最常用為inner join，資料表會留下連接欄位值相同的紀錄，另外還有left join與right join。
  • where：用於設定選取條件，當條件超過一個以上，並須使用and、or進行串接。
  • order by：用於設定選取排序，預設為遞增排序，可以設定關鍵字desc來進行遞減排序。
  • limit：用於設定選取資料表資料筆數，如果使用方式為limit 資料筆數 代表選取開始幾筆的資料筆數，如果設定limit 開始記錄位置,資料筆數 代表選取從特定紀錄位置開始的資料筆數。
  • group by：用於設定資料群組，having用於設定群組欄位條件。 


select distinct 欄位 as 新欄位名稱 from 資料表1 inner join 資料表2 on 資料表1.欄位1=資料表2.欄位1
and 資料表1.欄位2=資料表2.欄位2 where 條件1 and 條件2 order by 欄位1 desc limit 資料筆數

select count(欄位) as 新欄位名稱 group by 欄位 having 條件
--------------------------------------------------------
$stmt=$pdo->prepare('select * from iusers');
$stmt->execute();
$rows=$stmt->fetchAll();

常用函數包含sum、avg、count、min、max、is null等。

insert命令執行資料新增，第一個命令代表新增值至對應欄位，如果要使用參數進行新增時，PDO prepare新增命令使用?代表代入參數，而PDO execute命令則使用execute(array(代入值1,代入值2))方式代入參數。 


insert into 資料表(欄位1,欄位2) values(值1,值2)

insert into 資料表(欄位1,欄位2) values(?,?)
----------------------------------------------
$stmt=$pdo->prepare('insert into iusers(account,password,nickname,email,image) values(?,?,?,?,?)');
$stmt->execute(array('a','123','nice','a@stust.edu.tw','a.jpg'));

update命令執行資料更新，第一個命令代表更新值至對應欄位，如果要使用參數進行新增時，PDO prepare更新命令使用?代表代入參數，而PDO execute命令則使用execute(array(代入值1,代入值2))方式代入參數。 


update 資料表 set 欄位1=值1,欄位2=值2 where 條件

update 資料表 set 欄位1=?,欄位2=? where 條件
----------------------------------------------
$stmt=$pdo->prepare('update iusers set email="b@gmail.com"'); //請思考命令執行後有何問題
$stmt->execute();

delete命令執行資料刪除，第一個命令代表更新值至對應欄位，如果要使用參數進行新增時，PDO prepare更新命令使用?代表代入參數，而PDO execute命令則使用execute(array(代入值1,代入值2))方式代入參數。 


delete from 資料表 where 條件
